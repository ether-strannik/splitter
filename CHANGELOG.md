### Added
- Panorama slicer utility for cutting large images into printable 8.5x11 landscape pages
- Dynamic page width calculation based on image height ratio
- Right-to-left slicing mode (start from right edge)
- Custom start point selection via right-click
- Print dialog with printer selection dropdown
- Page range options (all, visible, custom range)
- Fit-to-page printing with Windows print integration
- Pan navigation (mouse drag, arrow keys)
- Zoom (scroll wheel)
- Page grid overlay with page numbers
- Export single page (click), all pages, or visible pages as PNG
